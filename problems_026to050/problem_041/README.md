## 41. Pandigital prime

### Translation
* [한국어](./translation-ko.md)

We shall say that an <var>n</var>-digit number is pandigital if it makes use of all the digits 1 to <var>n</var> exactly once. For example, 2143 is a 4-digit pandigital and is also prime.
